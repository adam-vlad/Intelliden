# Intelliden 🏠
Smart home ecosystem controllable via Mobile/Tablet, AR Headset, and Smartwatch.

> HCI Project — FII UAIC, Year 3

## What is Intelliden?
You are a specialized engineer passionate about home automation. You build a smart home 
ecosystem for your living space, controllable via smartphone/tablet, smartwatch, and AR glasses. 
Features include: adding new devices, viewing statistics, setting automations and gestures, 
and interacting co-op with family or roommates.
